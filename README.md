# GitHub Pages site

This folder is a ready-to-publish static site for App Store URLs.

## Files

- `index.html`
- `privacy.html`
- `support.html`

## Replace before publishing

- Update `support@yourdomain.com` in `support.html` to your real support email.
- If needed, adjust the app name in `privacy.html` and `support.html`.

## Publish with GitHub Pages

1. Create a new public repository on GitHub.
2. Copy these files to the repository root.
3. Open repository `Settings` > `Pages`.
4. Set `Build and deployment` to `Deploy from a branch`.
5. Choose the `main` branch and `/root` folder.
6. Save.

After publishing, use:

- Privacy policy URL: `https://<username>.github.io/<repo>/privacy.html`
- Support URL: `https://<username>.github.io/<repo>/support.html`
